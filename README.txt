A-I Speak! 디자인 패키지

구성:
index.html               -- 첫 화면 (인트로)
sub/subpage.html         -- 언어/단계/카테고리 선택 페이지 (디자인)
style/common.css         -- 공통 스타일
style/index.css          -- 페이지별 스타일
js/main.js               -- 간단한 UI 상호작용(디자인용)
assets/images/placeholder.svg -- 카메라가 없을 때 표시되는 플레이스홀더 이미지

사용법:
1) 압축을 풀고 웹서버 루트(또는 파일://로도 일부 기능 확인 가능)에 올리세요.
2) index.html을 열고 '▶' 버튼을 눌러 디자인 흐름을 확인하세요.
3) 실제 기능(카메라, 음성녹음, 백엔드 연동)은 포함되어 있지 않습니다. js/main.js는 디자인 데모용으로 시뮬레이션을 포함합니다.
